# 死亡主宰 (Dead Master)

死亡主宰是在草地区域出现的强大Boss，拥有两个截然不同的战斗阶段。挑战此Boss需要玩家之间进行有效的沟通与协作。

## 第一阶段：虚影形态

战斗开始时，死亡主宰以一个无法被伤害的虚影形态出现。在此阶段，它不会移动或直接攻击，而是通过召唤一系列爪牙来考验挑战者。玩家的目标是清除所有被召唤的生物，以迫使Boss进入下一阶段。

| 属性 | 数值 / 描述 |
| :--- | :--- |
| **等级** | 30 |
| **生命值** | 1000 (此阶段无敌) |
| **护甲** | 10 |
| **伤害** | 0 |
| **外观** | 玩家 `Tsuki1509` 的模型 |
| **特性** | 无敌, 静止, 无碰撞体积, 无重力 |

### 阶段技能

*   **技能 1：战斗开启**
    *   **触发条件**：当有玩家进入Boss周围4格范围时首次触发。
    *   **触发周期**：每场战斗仅触发一次。
    *   **技能描述**：Boss开始缓慢浮空，并发出凋灵生成的咆哮声，宣告战斗正式开始。

*   **技能 2：爪牙召唤**
    *   **触发条件**：战斗开始后。
    *   **触发周期**：大约每5分钟触发一次。
    *   **技能描述**：Boss举起武器，在短暂咏唱后，于半径50格内的所有玩家身边召唤第一波怪物，包括 **6只骷髅爪牙** 和 **4只凋灵骷髅爪牙**。

*   **技能 3：精锐召唤**
    *   **触发条件**：当第一波召唤的“骷髅爪牙”和“凋灵骷髅爪牙”被全部消灭后触发。
    *   **触发周期**：接收到信号后触发一次。
    *   **技能描述**：Boss进行一次更长时间的咏唱，随后在半径50格内的所有玩家身边召唤第二波更强大的怪物，包括 **3只骷髓** 和 **3只髑髅**。

*   **技能 4：阶段转换**
    *   **触发条件**：当第二波召唤的“骷髓”和“髑髅”被全部消灭后触发。
    *   **触发周期**：接收到信号后触发一次。
    *   **技能描述**：Boss发出愤怒的咆哮，在持续的音效和粒子效果后，一道闪电劈中自身，虚影形态消失，**死亡主宰的真身** 将在原地出现，战斗进入第二阶段。

## 召唤物详解

### 骷髅爪牙
*   **外观**：手持铁剑的普通骷髅。
*   **等级**：3
*   **生命值**：30
*   **伤害**：1
*   **掉落物**：少量经验值和狐币。

### 凋灵骷髅爪牙
*   **外观**：手持钻石剑的凋灵骷髅。
*   **等级**：5
*   **生命值**：50
*   **伤害**：1
*   **技能**：攻击会为目标附加持续5秒的 **凋灵 III** 效果。
*   **掉落物**：少量经验值、狐币，并有7%的几率掉落凋灵骷髅头颅。

### 骷髓
*   **外观**：空手的普通骷髅。
*   **等级**：10
*   **生命值**：100
*   **伤害**：0 (通过技能造成伤害)
*   **技能**：周期性地对自身周围3格范围内的所有玩家造成6点伤害，并伴有灵魂火焰粒子效果。
*   **掉落物**：经验值、狐币、残魂(5%)、玉化骨骼(10%)，并有1%的几率掉落骷髅头颅。

### 髑髅
*   **外观**：一个隐形的盔甲架顶部佩戴着凋灵骷髅头颅，表现为一个悬浮的、燃烧着火焰的头颅。
*   **等级**：15
*   **生命值**：100
*   **伤害**：20
*   **技能 1**：近战攻击会点燃目标，持续5秒。
*   **技能 2**：每15秒，它会短暂浮空并向目标连续发射4颗凋灵头颅射弹。
*   **掉落物**：经验值、狐币、残魂(5%)、玉化骨骼(10%)，并有20%的几率掉落凋灵骷髅头颅。

## 第二阶段：真身形态

当所有召唤物被清除后，死亡主宰的真身将会降临。此阶段的Boss拥有极高的攻击性，并且会随着生命值的降低解锁更多毁灭性的技能。

| 属性 | 数值 / 描述 |
| :--- | :--- |
| **等级** | 30 |
| **生命值** | 1000 |
| **护甲** | 10 |
| **伤害** | 30 |
| **外观** | 玩家 `Tsuki1509` 的模型 |
| **特性** | 可被伤害, 移动速度0.2, 无法被击退 |

### 阶段技能

*   **技能 1：冲锋斩**
    *   **触发条件**：进入第二阶段后。
    *   **触发周期**：约每13秒一次。
    *   **技能描述**：Boss锁定一名仇恨最高的玩家，发出一声音效后高速冲向目标，抵达后立即进行三次范围为4格的挥砍，每次造成25点伤害。

*   **技能 2：场地改造**
    *   **触发条件**：生命值低于700后解锁。
    *   **触发周期**：约每5分钟一次。
    *   **技能描述**：Boss咏唱后，将自身周围半径32格内的所有地面方块变为黏液块，持续100秒。同时，范围内的所有玩家会获得持续15秒的 **中毒 I** 效果。在此期间，Boss自身获得 **抗性提升 IV**。

*   **技能 3：毁灭剑气**
    *   **触发条件**：生命值低于400后解锁。
    *   **触发周期**：约每30秒一次。
    *   **技能描述**：Boss发出蓄力音效并开始旋转，随后向外释放数道不断扩大的环形剑气，最大半径可达14格，被剑气命中的玩家会受到20点伤害。

*   **技能 4：浮空轰炸**
    *   **触发条件**：生命值低于300后解锁。
    *   **触发周期**：约每25秒一次。
    *   **技能描述**：Boss浮空并向战场投下共计9枚**魔法炸弹**。炸弹落地后会发出持续的警告音效，并在约3秒后爆炸，对6格范围内的玩家造成30点伤害。

## 战斗失败条件
如果在战斗的任意阶段，距离Boss 50格内没有任何存活的玩家，Boss将会消失，战斗宣告失败，并发出一条全服广播。

## 战利品
成功击败死亡主宰后，会在其死亡位置生成一个金块状的奖励实体。
*   **领取方式**：所有参与战斗的玩家均可右键点击该实体，领取一份战利品。**请确保背包有空位**。
*   **奖励内容**：右键点击后，玩家会获得一个“死亡主宰战利品箱”。将其放置并右键打开后可获得以下奖励：

| 物品 | 数量 | 掉落几率 |
| :--- | :--- | :--- |
| **狐币** | 600 - 610 | 100% |
| **死灵髓晶** | 1 | 100% |
| **下界之星** | 1 | 50% |
| **附魔书 (亡灵杀手 VI)** | 1 | 50% |
